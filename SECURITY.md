# Security Policy

## Supported Versions

We actively monitor and fix security issues in the following versions:

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please do **not** open a public issue on GitHub. Instead, report it privately to the maintainer:

- **Email**: masud50cf@gmail.com
- **Response Time**: We aim to acknowledge reports within 48 hours and provide a resolution plan within 7 days.

Please include the following details in your report:
- A description of the vulnerability.
- A proof of concept or steps to reproduce the issue.
- Potential impact on users or the deployment environment.

Thank you for helping keep this project secure!
